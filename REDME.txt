==================================

  arriere-el-godem/

  ├── images/             ← Put all your photos here
  │   ├── IMG_6215.jpg    → Hero (full-screen background)
  │   ├── IMG_6222.jpg    → About / Premier Drop section
  │   ├── IMG_6237.jpg    → Featured Product card

  └── README.txt          ← This file

HOW TO USE:
1. Copy all your photos into the "images/" folder
2. Make sure filenames match exactly (case-sensitive)
3. Open index.html in any browser — images appear instantly

NOTES:
- If an image is missing, the slot shows a dark placeholder (no broken icon)
- The website is fully responsive (mobile, tablet, desktop)
- No internet connection required (fonts load from Google Fonts)
